Installation
************

Installation
############

Install with `pip`:

::

    $ sudo apt-get install libffi-dev libssl-dev python-dev
    $ pip3 install peerplays

Manual installation:

::

    $ git clone https://github.com/xeroc/python-peerplays/
    $ cd python-peerplays
    $ python3 setup.py install --user

Upgrade
#######

::

   $ pip install --user --upgrade
